# Chat App - Angular

## Setup

npm install angular
npm install -g angular-cli
cd client
npm install

## Server running
cd server

node chatserver.js

http://localhost:4200/login

eftir að þú loggast inn ferðu inn í rooms

http://localhost:4200/rooms

## Client
cd client

ng serve


The style needs to fulfill ng lint, so remember to run that now and then!
